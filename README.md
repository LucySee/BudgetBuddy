# 💸 BudgetBuddy

> Know your daily budget. Spend with confidence.  
> 每天花多少，心里有数。  
> Tahu bajet harian anda, belanja dengan bijak.

**[👉 Open BudgetBuddy](https://lucysee.github.io/BudgetBuddy/)**

Available in English, 简体中文, and Bahasa Melayu.

---

## ✨ Features

- **Daily budget** — know exactly how much you can spend today
- **High budget days** — set selected days to spend a % more (e.g. weekends)
- **Savings tracker** — underspend today, the savings are truly yours
- **Overspend safety net** — if you go over, excess spreads across remaining days
- **Expense logging** — log by category (Need / Want / Impulse) and sub-category
- **Edit & delete** — tap any log entry to modify or remove
- **Monthly summary** — spending pace, top category, impulse ratio, streak & breakdown
- **Multi-currency** — RM, SGD, USD, EUR and more, or enter custom symbol
- **3 languages** — English, Chinese, Malay with native insight messages

---

## 📊 Monthly Summary

| Metric | Description |
|--------|-------------|
| 🏦 Daily Savings | Total underspend vs fixed daily budget |
| 💰 Spent This Month | Total vs monthly budget % |
| 📦 Top Category | Most frequent expense category |
| ⚡ Impulse Ratio | % of expenses marked as impulse |
| 🔥 Streak | Consecutive days without overspending |
| 📂 Sub-category | Spending breakdown with progress bars |
